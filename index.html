About This Bot

This is a personal Discord bot that retrieves and displays public Etsy product information for reference and tracking purposes. The bot uses the Etsy Open API to fetch publicly available data about products and shops.

How It Works

The bot operates within Discord servers, allowing users to issue commands to retrieve Etsy product information. All data is sourced from Etsy's public API endpoints using read-only GET requests. The bot does not store any personal user data or modify any Etsy listings.

Users can paste Etsy product URLs to get detailed information, or set up shop tracking to monitor specific Etsy shops for new product listings. All notifications are sent directly to designated Discord channels.

Technical Stack

Python Discord.py Etsy Open API v3 REST API JSON

Privacy & Data Usage

This bot only accesses publicly available Etsy product information through official API endpoints. It does not:

  • Store or collect personal user data
  • Modify or create Etsy listings
  • Make purchases or transactions
  • Sell or redistribute data to third parties
  • Access private or restricted information
